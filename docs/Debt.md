# Debt

The Debt page tracks outstanding debts (credit cards, loans, mortgages, etc.) and projects their repayment over time.

## Adding a Debt

Click **Add Debt** to open the form. Fill in the fields and click **Save**.

## Fields

| Field | Description |
|---|---|
| **Name** | A label for the debt (e.g. "Visa Card", "Car Loan"). |
| **Balance** | The current outstanding balance in pounds. |
| **APR (%)** | Annual Percentage Rate. Used to calculate monthly interest. |
| **Minimum Payment** | The minimum monthly payment required by the lender. |
| **Overpayment** | Any additional amount paid above the minimum each month. |
| **Compounding** | How often interest compounds: Monthly or Daily. |
| **Posting Day** | The day of the month the payment is due. |
| **Household Debt** | Tick to include this debt in Household summary views. |
| **Notes** | Optional free-text notes. |

## Deal Periods

Deal Periods allow you to track promotional interest rates (e.g. 0% balance transfer offers). Each deal period has:

| Field | Description |
|---|---|
| **Deal Name** | A label for the deal (e.g. "0% BT Offer"). |
| **Interest Rate (%)** | The promotional APR during this period. |
| **Start Date** | When the promotional rate begins. |
| **End Date** | When the promotional rate expires. |
| **Email Reminder** | Enable to receive an email reminder before the deal expires. |
| **Reminder Days** | How many days before expiry to send the reminder. |

Deal periods override the standard APR for their active date range in repayment calculations.

## Repayment Schedule

Click **View Schedule** on any debt to see a month-by-month projection table:

| Column | Description |
|---|---|
| **Month** | The calendar month. |
| **Opening Balance** | Balance at the start of the month. |
| **Interest** | Interest charged during the month. |
| **Payment** | Total payment made (minimum + overpayment). |
| **Closing Balance** | Balance at the end of the month. |

## Payoff Summary

Below the schedule, a summary shows:

- **Months remaining** until the debt is paid off
- **Total interest** that will be paid over the repayment period
- **Projected payoff date**
