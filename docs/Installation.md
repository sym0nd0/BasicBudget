# Installation

BasicBudget can be installed in two ways: via Docker (recommended for production and self-hosting) or manually (for development or environments without Docker).

## Installation Methods

| Method | Best For |
|---|---|
| [[Docker Setup\|Docker-Setup]] | Production deployments, self-hosting, NAS, VPS |
| [[Manual Setup\|Manual-Setup]] | Local development, contributing to the project |

## Before You Start

Review the [[System Requirements\|System-Requirements]] page to confirm your environment meets the minimum requirements.

## Configuration

After installation, refer to the [[Configuration]] page for a full list of environment variables and their defaults.

## Updating

See [[Updating BasicBudget\|Updating-BasicBudget]] for instructions on pulling new versions and applying database migrations.
