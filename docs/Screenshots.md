# Screenshots

All screenshots show BasicBudget in both Light and Dark themes.

## Dashboard

The Dashboard provides an at-a-glance view of your monthly financial position, including summary cards, income vs outgoings bar chart, and expense breakdown donut.

The Income page lists all active income entries for the selected month, with sortable columns and recurrence details.
