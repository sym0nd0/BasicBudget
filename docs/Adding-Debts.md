# Adding Debts

## Adding a Debt

1. Go to **Debt** in the sidebar.
2. Click **Add Debt**.
3. Fill in the fields and click **Save**.

## Fields

| Field | Description |
|---|---|
| **Name** | A label for the debt (e.g. "Visa Card", "Car Loan"). |
| **Balance** | The current outstanding balance in pounds. |
| **APR (%)** | Annual Percentage Rate. Used to calculate monthly interest. |
| **Minimum Payment** | The minimum monthly payment required by the lender. |
| **Overpayment** | Any additional amount paid above the minimum each month. |
| **Compounding** | How often interest compounds: **Monthly** or **Daily**. |
| **Posting Day** | The day of the month the payment is due. |
| **Household Debt** | Tick to include this debt in Household summary views. |
| **Notes** | Optional free-text notes. |

## Compounding

- **Monthly compounding** — interest is calculated once per month on the outstanding balance.
- **Daily compounding** — interest accrues daily based on the annual rate divided by 365.

Most credit cards and personal loans in the UK use daily compounding. Check your lender's terms if unsure.

## Updating the Balance

As you make payments, update the **Balance** field to reflect the current outstanding amount. BasicBudget uses this balance as the starting point for all repayment projections.

## Editing and Deleting

Click the **edit icon** on any debt card to update it. Click the **delete icon** to remove it permanently. Deleted debts cannot be recovered.
