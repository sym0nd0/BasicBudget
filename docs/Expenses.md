# Expenses

The Expenses page lists all expense entries active for the selected month. Expenses can be fixed (predictable amounts) or variable (amounts that change month to month).

## Adding an Expense

Click **Add Expense** to open the form. Fill in the fields and click **Save**.

## Fields

| Field | Description |
|---|---|
| **Name** | A label for the expense (e.g. "Rent", "Groceries"). |
| **Amount** | The expense amount in pounds. |
| **Posting Day** | The day of the month the expense is due (1–31). |
| **Category** | The expense category (e.g. Housing, Food, Transport). Categories are managed in the [Admin panel](Admin.md). |
| **Type** | **Fixed** (same amount each month) or **Variable** (amount varies). |
| **Account** | The payment account used for this expense. |
| **Recurrence** | How often the expense occurs: One-off, Monthly, Weekly, or Fortnightly. |
| **Split Ratio** | For household expenses, the percentage split between members (e.g. 50/50). |
| **Start Date** | The first month this expense is active. |
| **End Date** | The last month this expense is active. |
| **Household Expense** | Tick to include this expense in Household summary views. |
| **Notes** | Optional free-text notes. |

## Fixed vs Variable

- **Fixed** expenses have a set amount that does not change. They appear with their amount in full every active month.
- **Variable** expenses are flagged as unpredictable. Their amount can be updated each month to reflect actual spending.

## Household Expenses and Split Ratio

When an expense is marked as a **Household Expense**, a split ratio field becomes available. This allows you to record how the cost is divided between household members (e.g. 60% / 40%). The split is informational and used in Household summary calculations.

## Table Features

- **Sortable columns** — click any column header to sort the list.
- **Expandable rows** — click a row to expand it and see the full notes and details.

## Editing and Deleting

Click the **edit icon** on any row to update an entry. Click the **delete icon** to remove it permanently.
