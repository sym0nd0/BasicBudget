# Expenses

The Expenses page lists all expense entries active for the selected month.

## Adding an Expense

Click **Add Expense** to open the form. Fill in the fields and click **Save**.

## Fields

| Field | Description |
|---|---|
| **Name** | A label for the expense (e.g. "Rent", "Groceries"). |
| **Amount** | The expense amount in pounds. |
| **Posting Day** | The day of the month the expense is due (1–31). |
| **Category** | The expense category (e.g. Housing, Food, Transport). Categories are managed in the [Admin panel](Admin). |
| **Account** | The payment account used for this expense. |
| **Recurrence** | How often the expense occurs: One-off, Monthly, Weekly, Yearly, or Fortnightly. |
| **Split Ratio** | For household expenses, the app currently applies an equal split across household members. |
| **Start Date** | The first date this expense is active. |
| **End Date** | The last date this expense is active. |
| **Household Expense** | Tick to include this expense in Household summary views. |
| **Notes** | Optional free-text notes. |

## Household Expenses and Split Ratio

When an expense is marked as a **Household Expense**, the app applies an equal split across all household members and uses your share in summary totals and comparison indicators.

## Table Features

- **Sortable columns** — click any column header to sort the list.
- **Expandable rows** — click a row to expand it and see the full notes and details.

## Editing and Deleting

Click the **edit icon** on any row to update an entry. Click the **delete icon** to remove it permanently.
