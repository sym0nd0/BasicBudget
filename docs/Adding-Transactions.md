# Adding Transactions

BasicBudget organises financial data into four types of entries. Each has its own page and form.

## Income

Income entries represent money coming in — salary, freelance work, rental income, benefits, or any other source.

- Go to **Income** → **Add Income**
- Key fields: Name, Amount, Posting Day, Recurrence, Gross/Net toggle
- See [[Income]] for the full field reference

## Expenses

Expense entries represent money going out — rent, bills, groceries, subscriptions, and any other cost.

- Go to **Expenses** → **Add Expense**
- Key fields: Name, Amount, Category, Type (Fixed/Variable), Account, Recurrence
- See [[Expenses]] for the full field reference

## Debts

Debt entries track outstanding balances on credit cards, loans, and mortgages, with interest calculations and repayment projections.

- Go to **Debt** → **Add Debt**
- Key fields: Name, Balance, APR, Minimum Payment, Overpayment, Compounding
- See [[Adding Debts\|Adding-Debts]] for the full field reference

## Savings Goals

Savings goal entries track progress towards financial targets such as an emergency fund, holiday, or home deposit.

- Go to **Savings** → **Add Goal**
- Key fields: Name, Target Amount, Current Amount, Monthly Contribution, Target Date
- See [[Savings]] for the full field reference

## Common Fields

All four entry types share some common fields:

| Field | Description |
|---|---|
| **Household** | Tick to include this entry in the shared Household view. |
| **Notes** | Optional free-text notes for your own reference. |
| **Start Date** | The first month this entry is active (`YYYY-MM`). Leave blank to start immediately. |
| **End Date** | The last month this entry is active. Leave blank for ongoing. |
