# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for BasicBudget. Each ADR documents a significant technical decision, its context, and its consequences.

## Template

Use [ADR-000-template.md](ADR-000-template.md) as the starting point for new ADRs. To create a new ADR, run `/adr <title>` in Claude Code.
